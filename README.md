# JAM2-Analyzer

このリポジトリは JAM2 を使って重イオン衝突の解析を行うためのコードです。

## セットアップ手順
INSTALL_jp.md を参照してください。  

ビルドされた JAM2 実行ファイルは以下に配置されます。
- `jam2-code/install/bin/jam`

## このリポジトリの構成
- `jam2-code/`: 本家 JAM2 リポジトリ（Git Submodule）
- `input/`: 入力ファイル
- `patches/`: 最新環境でビルドするための互換性パッチ
- `setup.sh`: 全自動ビルドスクリプト
- `scripts/`: 解析用スクリプト（順次追加予定）

## 実行方法

```bash
./jam2-code/install/bin/jam -f input/30gev_pAu.inp
```

## 実行結果
jamを実行するとphase.dat.gzというファイルが生成される。これが計算結果。

### **1. 全体ヘッダー (ファイルの先頭1行)**

1行目には、シミュレーション全体の基本情報が記載されている。

- `# nev`: 総イベント数
- `ycm`: 重心系のラピディティ (Rapidity)
- `gammaA`, `gammaB`: ビーム粒子 A, B のガンマ（Lorentz）因子

### **2. イベントヘッダー (各イベントの開始行)**

`#` で始まる行が新しいイベントの開始を示す。そのイベントの統計情報が並んでいる。

- `iev`: イベント番号 (1, 2, ...)
- `nv`: そのイベントにおける全粒子の数 (Multiplicity)
- `ncollG` / `npartG`: Glauberモデルで予測された衝突数 / 参加核子数
- `b`: **インパクトパラメータ (衝撃係数) [fm]**
- **`npart`**: 実際に衝突に参加した核子の総数
- **`ncoll`**: 実際に発生した全衝突回数（カスケード内での総衝突数）
- `ncollBB`: そのうちバリオン・バリオン間の衝突数

### **3. 粒子データ**

イベントヘッダーに続き、そのイベントで生成された粒子が1行ずつ並ぶ。 13列に分かれており、主な意味は以下の通り：

| **列番号** | **項目名** | **意味** |
| --- | --- | --- |
| 1 | **status** | 粒子の状態（1なら安定な終状態） |
| 2 | **`pid`** | 粒子ID (**PDGコード**)。例：2212=陽子, 211=π+, -211=π- |
| 3 | **`ncoll`** | その粒子が経験した累積衝突回数 |
| 4 | **`mass`** | 粒子の質量 [GeV/$c^2$] |
| 5-7 | **`px, py, pz`** | 粒子の運動量成分 [GeV/$c$] |
| 8 | **`energy`** | 粒子のエネルギー $E$ [GeV] |
| 9-11 | **`x, y, z`** | 粒子の座標成分 [fm] |
| 12 | **`time`** | 粒子の時刻 $t$ [fm/$c$] |
| 13 | **`tf`** | Freeze-out時間（最終衝突点などの時刻）[fm/$c$] |

---

このファイルがあれば、各イベントでどの方向にどのくらいの速さで粒子が飛んでいったか、どこで最後に相互作用したかといった情報をすべて抽出して解析することができる。

## phase.dat.gzファイルからrootファイルへの変換

```bash
python3 scripts/phase_to_root.py 
```

実行すると、phase.dat.gzをインプットにして、rootfile/phase.rootというファイルが生成される。これが計算結果のルートファイル。

### phase.root の中身

生成されたphase.rootファイルには、`tree` という名前の単一のTTreeが含まれており、イベント全体に関する情報と、そのイベントに含まれる粒子のリスト情報（配列）がまとめて格納されています。

#### TTree: `tree`

**イベント変数（スカラ）**

| **変数名** | **型** | **説明** | **単位** |
|---|---|---|---|
| **iev** | int | イベント番号 | - |
| **nv** | int | そのイベントにおける全粒子の数 | - |
| **b** | double | インパクトパラメータ (衝撃係数) | fm |
| **npart** | int | 実際に衝突に参加した核子の総数 | - |
| **ncoll** | int | 実際に発生した全衝突回数（カスケード内での総衝突数） | - |

**粒子変数（配列: `std::vector`）**
各イベントに含まれる `nv` 個の粒子の情報がそれぞれ配列として格納されています。

| **変数名** | **型** | **説明** | **単位** |
|---|---|---|---|
| **status** | vector<int> | 粒子の状態（1なら安定な終状態） | - |
| **pid** | vector<int> | 粒子ID (PDGコード) | - |
| **p_ncoll**| vector<int> | その粒子が経験した累積衝突回数 | - |
| **mass** | vector<double>| 粒子の質量 | GeV/$c^2$ |
| **px** | vector<double>| 粒子の運動量成分 $p_x$ | GeV/$c$ |
| **py** | vector<double>| 粒子の運動量成分 $p_y$ | GeV/$c$ |
| **pz** | vector<double>| 粒子の運動量成分 $p_z$ | GeV/$c$ |
| **energy** | vector<double>| 粒子のエネルギー $E$ | GeV |
| **x** | vector<double>| 粒子の座標成分 $x$ | fm |
| **y** | vector<double>| 粒子の座標成分 $y$ | fm |
| **z** | vector<double>| 粒子の座標成分 $z$ | fm |
| **t** | vector<double>| 粒子の時刻 $t$ | fm/$c$ |
| **tf** | vector<double>| Freeze-out時間（最終衝突点などの時刻） | fm/$c$ |
